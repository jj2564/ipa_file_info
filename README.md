# ipa_file_info
This is for enterprise distribution. It may show the bundle id, build xcode version, expiration date ... 

## Enviroment
Python: 2.7
OS: mac os

## How to use
```script
$python -f {file_path}
```

## Result

```
APP_FILE_NAME
-----------------------------------------
App Name           : APP_NAME
Expiration Date    : 2020-03-26
Bundle ID          : YOUR_BUNDLE_ID
Minimum Support    : iOS 9.0
Build Xcode        : 1120
Creation Date      : 2019-03-27
```
